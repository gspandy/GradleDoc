起航¶

下面的一组指南将向你介绍Gradle的基础知识，帮助没有基础的你进入Gradle世界。

第四章， 安装Gradle ::
教你如何安装Gradle。
第五章， 构建脚本基础 ::
介绍构建脚本的两个基本元素：项目(projects)和任务(tasks)。
第六章， Java项目快速上手 ::
介绍如何用Gradle来构建Java项目。
第七章， 依赖管理基础 ::
介绍如何使用Gradle的依赖管理。
第八章， 构建Groovy项目快速上手 ::
用Gradle来构建Groovy项目。
第九章， 构建Web项目迅速上手 ::
使用Gradle来构建Web应用程序。
